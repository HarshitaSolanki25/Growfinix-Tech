# Growfinix Tech Internship

## Task 1: Titanic Survival Analysis

### Project Overview

This project analyzes the Titanic dataset to identify the factors that influenced passenger survival. The dataset was cleaned, processed, and explored using Python data analysis libraries.

### Objective

* Clean and preprocess the dataset.
* Handle missing values.
* Analyze survival patterns among passengers.
* Visualize important relationships using charts.
* Generate meaningful insights from the data.

### Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
* GitHub

### Analysis Performed

* Data Cleaning and Preprocessing
* Missing Value Treatment
* Survival Distribution Analysis
* Gender-wise Survival Analysis
* Passenger Class Analysis
* Age Distribution Analysis
* Family Size Impact on Survival
* Fare vs Survival Analysis
* GroupBy and Pivot Table Insights

### Key Findings

* Female passengers had a significantly higher survival rate than male passengers.
* First-class passengers were more likely to survive than second and third-class passengers.
* Higher ticket fares were generally associated with better survival chances.
* Passengers traveling with small families had better survival rates.
* Gender and passenger class were the most influential factors affecting survival.

The analysis demonstrates how data cleaning, visualization, and exploratory data analysis can be used to uncover meaningful insights from real-world datasets. The project highlights the importance of passenger demographics and socio-economic factors in determining survival outcomes during the Titanic disaster.
